# Personal Portfolio | Lakkoju Ganesh Koushik

A modern, interactive personal portfolio website showcasing my experience in 3D Reconstruction, XR Development, and Software Engineering. 

Built with a focus on high-performance rendering and premium UI/UX, featuring a custom glassmorphism design system and dynamic light/dark mode environments.

## 🚀 Tech Stack

* **Core Framework:** React.js powered by Vite for lightning-fast HMR and building.
* **Styling:** Tailwind CSS (utilizing class-based dark mode and backdrop-blur utilities).
* **Animations:** Framer Motion for smooth route transitions, hover states, and dynamic background rendering.
* **Deployment:** CI/CD pipeline integrated with Vercel.

## ✨ Key Features

* **Frosted Glass UI:** Premium glassmorphism aesthetic with responsive, semi-transparent components.
* **Dynamic Theming:** Seamless switching between Light and Dark modes with persistent background state changes.
* **Interactive Backgrounds:** Custom-built animated floating orbs using Framer Motion to provide depth without heavy WebGL overhead.
* **Component-Based Architecture:** Modular design separating layout, pages, and interactive components.
